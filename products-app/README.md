# Premium Store - Products App

Bu loyiha Fake Store API dan mahsulotlarni olish va chiroyli UI bilan ko'rsatish uchun yaratilgan React Vite ilovasidir.

## Xususiyatlari

- ✨ **Chiroyli dizayn** - Zamonaviy gradient va animatsiyalar bilan
- 🔄 **Loading holati** - Skelaton komponentlar bilan
- ⚠️ **Xatolik boshqaruvi** - Chiroyli error UI bilan
- 🔍 **Popup modal** - Mahsulot tafsilotlari uchun
- 📱 **Responsive** - Barcha qurilmalarda ishlaydi
- ⭐ **Rating ko'rsatish** - Yulduzcha bilan baholash
- 🏷️ **Kategoriya belgilari** - Mahsulot turlari
- 🛒 **Interaktiv kartalar** - Hover effectlar bilan

## Ishlatilgan texnologiyalar

- React 19 + Vite
- Tailwind CSS 4
- Axios
- Fake Store API

## O'rnatish va ishga tushirish

1. Loyihani klonlash:
\`\`\`bash
git clone <repository-url>
cd products-app
\`\`\`

2. Bog'liqliklarni o'rnatish:
\`\`\`bash
npm install
\`\`\`

3. Development serverini ishga tushirish:
\`\`\`bash
npm run dev
\`\`\`

4. Build qilish:
\`\`\`bash
npm run build
\`\`\`

## Komponentlar tuzilishi

- `Products.jsx` - Asosiy komponent
- `ProductModal.jsx` - Mahsulot tafsilotlari popup
- `ui/Skeleton.jsx` - Loading skelaton
- `ui/ErrorWidget.jsx` - Xatolik ko'rsatish

## API

Loyiha [Fake Store API](https://fakestoreapi.com/products) dan foydalanadi.
