# User CRUD Application

Bu loyiha React + Vite va Tailwind CSS yordamida yaratilgan CRUD (Create, Read, Update, Delete) ilovasi.

## Xususiyatlar

- **Form (Chap tomonda)**: Foydalanuvchi ma'lumotlarini kiritish uchun
  - To'liq ism
  - Email manzil  
  - Parol
  - Tug'ilgan sana
  - Jins
  
- **Jadval (O'ng tomonda)**: Ro'yxatdan o'tgan foydalanuvchilarni ko'rsatish
  - Reytingga asoslanib tartiblangan
  - Har bir foydalanuvchi uchun Edit va Delete tugmalari
  - Random ball (points) tizimi

## Texnologiyalar

- React 18
- Vite
- Tailwind CSS
- useState Hook
- useRef Hook

## O'rnatish va ishlatish

1. Loyihani yuklab oling
2. Bog'liqliklarni o'rnating:
   ```bash
   npm install
   ```

3. Development serverini ishga tushiring:
   ```bash
   npm run dev
   ```

4. Brauzeringizda `http://localhost:5173` ga o'ting

## Qanday ishlaydi

1. **Create**: Formadagi barcha maydonlarni to'ldiring va "Submit" tugmasini bosing
2. **Read**: Barcha foydalanuvchilar jadvalda ko'rsatiladi
3. **Update**: Jadvaldagi "Edit" tugmasini bosganda ma'lumotlar formaga yuklanadi
4. **Delete**: "Delete" tugmasi foydalanuvchini o'chiradi

## Dizayn

- Responsive dizayn (mobil va desktop)
- Modern UI/UX
- Gradient background
- Hover effektlar
- Form validatsiya

Rasmda ko'rsatilgan kabi form chap tomonda, jadval o'ng tomonda joylashgan.
