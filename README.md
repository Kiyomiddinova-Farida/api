# AI Landing Page - React.js & SCSS

🚀 **Figma dizayniga asoslanib yaratilgan** zamonaviy AI platformasi uchun landing page. React.js va SCSS yordamida to'liq responsive va interactive komponentlar bilan amalga oshirilgan.

✨ **Rasimdagi Figma dizayni 100% aniq qayta yaratildi** - barcha animatsiyalar, ranglar va layoutlar bir xil.

## 🚀 Features

- **Modern Design**: Dark theme with blue accent colors and gradient elements
- **Responsive Layout**: Fully responsive design that works on all devices
- **Interactive Components**: 
  - Animated rocket with floating elements
  - 3D brain visualization with neural networks
  - Interactive carousel for awards section
  - Isometric data visualization
- **Smooth Animations**: CSS animations and transitions throughout
- **Component-Based Architecture**: Modular React components for easy maintenance
- **SCSS Styling**: Modern CSS with variables, mixins, and responsive design

## 🚀 Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Start Development Server**
   ```bash
   npm start
   ```

3. **Build for Production**
   ```bash
   npm run build
   ```
